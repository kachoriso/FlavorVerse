# FlavorVerse - ワインテイスティングノート生成アプリ

ワインのラベル写真とフレーバーホイール選択から、AIが詩的なテイスティングノートを自動生成するWebアプリです。

## 機能

- 📸 ワインラベルの写真撮影・アップロード
- 🎯 フレーバーホイールからの香味選択
- 🎭 ポエム度合いの調整（控えめ〜詩的）
- 📝 自動テイスティングノート生成

## 使い方

1. プロジェクトディレクトリで以下を実行:
```bash
npm run dev
```

2. ブラウザで http://localhost:8000 を開く

3. ワインラベルを撮影またはアップロード

4. フレーバーホイールから香味を選択

5. ポエム度合いを調整

6. テイスティングノートを生成

## 技術スタック

- バニラ JavaScript
- HTML5 (Camera API)
- CSS3
- 軽量なAI文章生成ロジック
